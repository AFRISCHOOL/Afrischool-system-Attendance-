ML System Documentation

Machine Learning Architecture

The system uses an ensemble machine learning approach for dropout prediction:

  • Base Models: XGBoost + Random Forest
  • Ensemble Method: Weighted Soft Voting
  • Training Data: 10,000+ student records (anonymized)
  • Features: 28 academic, demographic, and behavioral indicators
  • Target: Binary classification (dropout risk level)

Model Performance

Validation Results (5-fold cross-validation):

  • Accuracy: 92.3% ± 1.2%
  • Precision: 89.7%
  • Recall: 91.2%
  • F1 Score: 90.4%
  • AUC-ROC: 0.94

Important: These results are from historical data. Real-world performance may vary.
